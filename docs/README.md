# Quad Documentation

Quad is a university notice board and events web app: students browse official
notices and campus events, RSVP, bookmark, and comment; admins post and manage
everything through a dedicated admin area.

This is the documentation index. Each doc goes deep on one part of the system;
start with whichever matches what you're trying to do.

|---|---|
| [Architecture](./ARCHITECTURE.md) | Understand the stack, request flow, and how the pieces fit together |
| [Data Model](./DATA-MODEL.md) | Understand the database schema, relationships, and constraints |
| [Setup](./SETUP.md) | Get the project running locally from a clean clone |
| [Authentication](./AUTHENTICATION.md) | Understand login/register, sessions, roles, and route protection |
| [Features](./FEATURES.md) | See every feature, screen by screen, with the files behind each |

| [Scripts & Seeding](./SCRIPTS.md) | Look up what each `npm run` script does |
| [Deployment](./DEPLOYMENT.md) | Ship this to production |
| [Troubleshooting](./TROUBLESHOOTING.md) | Diagnose a specific recurring error |
